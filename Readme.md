## О проекте
Этот проект — мой pet-project и одновременно личный инструмент для учета инвестиций.

## Цель
Перейти от Excel к структурированной системе на PostgreSQL с возможностью дальнейшей аналитики и визуализации.

## Технологии
- PostgreSQL
- Docker
- SQL

## Текущая структура
Проект включает таблицы:
- accounts — инвестиционные счета
- assets — финансовые инструменты
- trades — сделки покупки и продажи
- cash_transactions — денежные операции по счету
